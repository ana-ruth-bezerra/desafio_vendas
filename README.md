## Previsões de vendas

O presente projeto tem como objetivo a resolução do desafio proposto pela plataforma Rocketseat, no módulo de Aprendizado de Máquina do curso de Inteligência Artificial. O desafio consiste na elaboração de um modelo de aprendizado de máquina capaz de realizar a previsão de vendas de uma loja, a partir de um dicionário de dados. Para isso foi solicitado:
- Tratamendo e formatação dos dados;
- Treinamento do modelo;
- Previsão das vendas do mês de dezembro;
- Montagem de histograma dos dados e plot de scatter.

Bibliotecas utilizadas:
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
